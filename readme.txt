Have a technical problem on your WordPress site? This plugin will give the answer. Or at least it tries to help you. :)

== Description ==
This plugin could help you when you stuck with a problem while creating a plugin, theme or something else. Based on the Hello Dolly plugin.

== Installation ==
1. Upload the folder `tps` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
